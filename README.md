# ConeStackerJ

## How to play
1) Install Java 21+ [Here](https://adoptium.net/temurin/releases?version=21&os=any&arch=any)
2) In a terminal, cd to the directory where ConeStackerJ-all.jar
3) Run java -jar ConeStackerJ.jar

ConeStackerJ is a fangame of [Cone Stacker](https://github.com/gavinskycastle/ConeStacker), but written in Java
